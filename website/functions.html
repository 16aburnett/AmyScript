Function Declaration Example 1

function int add (int a, int b)
{
    return a + b; 
}

A function is a useful tool used to break up a program into reusable subprograms. Similar to If-Statements and Loops, a function has a body of code that can be executed, however, a function's code can be executed whenever you Call the function.

Calling a Function: Calling a function tells the computer to run the code in the body of the function.

Functions are very useful because they act as their own contained program, meaning that any variables that you declare in the function's body will not affect the code outside of the body. Functions also accept Parameters (or input variables), and they have a Return Value.

Declaring Functions (function)

Creating a function requires 4 main components:

  • A return type
  • The function's name
  • A list of parameters (comma separated)
  • A code body

Below is the syntax for a function declaration in AmyScript.

Function Declaration Syntax

// Without Parameters
function <return-type-specifier> <function-name> ()
{
    <statement>
    <statement>
    ...
}
// With Parameters
function <return-type-specifier> <function-name> (<param0-type-specifier> <param0-name>, <param1-type-specifier> <param1-name>, ... <paramN-type-specifier> <paramN-name>)
{
    <statement>
    <statement>
    ...
}

Function Example

// This function takes two parameters, a and b, and returns the larger
function int max (int a, int b)
{
    if (a >= b) 
        return a;
    else
        return b;
}

Calling Functions

We now know how to create a function but how do we tell the computer to run the code in the function's body? With a function call!

A function call is a type of expression that allows you to tell the computer when to run a function and with what inputs (parameters).

A function call is very simple, all you need is the name of the function followed by parentheses (()). If a function requires parameters (inputs), then you can put them in between the parentheses separated by commas.

Function Call Syntax

// Without Parameters
<function-name> ()
// With Parameters - separated by commas 
<function-name> (<expression0>, <expression1>, ..., <expressionN>)

Function Call Example

// This function takes two parameters, a and b, and returns the larger
function int max (int a, int b)
{
    if (a >= b) 
        return a;
    else
        return b;
}
// Calling the above function:
int result = max (5, 7);
printInt (result);
println ("");

Result: 

7

In fact, printInt and println are also functions. They are built-in functions that are a part of AmyScript's Standard Library used for outputting information.

printInt requires an int as a parameter and it prints the given integer to the console.

println requires a char[] as a parameter and it prints the characters to the console followed by a newline character.

Both of those functions don't return anything and have Void as their return types.

Passing Parameters to Functions

Let's go more in depth about parameters.

Parameters are variables that you can pass to a function for it to use in its body.

Let's revisit the max function example. 

// This function takes two parameters, a and b, and returns the larger
function int max (int a, int b)
{
    if (a >= b) 
        return a;
    else
        return b;
}

The max function takes two integer parameters as input and based on the code in the function's body, it returns the value of the larger parameter. 

max (5, 7)

In the above function call, the 5 becomes the value of the first parameter (a) and the 7 becomes the value of the second (b).

Be Careful! The number of arguments and their types in the function call MUST match the number of parameters and their types in the function declaration.

Invalid Parameters Example

max (5)       // Error: Not enough parameters
max (5, 7, 3) // Error: Too many Parameters
max (10, 4.7) // Error: 4.7 (float) is not an int

Returning from Functions (return)

As mentioned previously, functions can return a value when they are done executing. The Return-Statement tells the computer what value to return. The Return-Statement also exits the function similar to a Break-Statement.

Return Statement Syntax

return <expression>;

In our max function example, we use the return statement to return either a or b, whichever is larger. 

// This function takes two parameters, a and b, and returns the larger
function int max (int a, int b)
{
    if (a >= b) 
        // function exits here with the value of a 
        return a;
    else
        // function exits here with the value of b 
        return b;
}

Since a function stops executing code when a return statement is reached, we can actually get rid of the else statement in our example. 

// This function takes two parameters, a and b, and returns the larger
function int max (int a, int b)
{
    if (a >= b) 
        // function exits here with the value of a 
        return a;
    // function exits here with the value of b 
    return b;
}
