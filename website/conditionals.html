The ability to do different things based on a given input condition is a crucial component of programming.

We've gone over comparison operators and logical operators in the Operators section, but they only return true or false values. We need a system to choose which code to run. AmyScript's answer to this is If-Statements.

If-Statements (if)

An If-Statement is a structure that runs a body of code only if a given condition evaluates to true.

If-Statement Syntax

if (<expression>) <expression>;
if (<expression>) 
{
    <statement>
    <statement>
    ...
}

If-Statement Example

int x = 7;
if (x > 0)
{
    println ("x is positive");
}
if (x == 0)
{
    println ("x is zero");
}
// curly braces are not needed for single statements 
if (x < 0)
    println ("x is negative");

Result:

x is positive

If-Else-Statements (else)

Sometimes you might want to do something else if a condition is false. You can achieve this by having another if-statement with the opposite condition as the first, however, that seems tedious.

AmyScript has an Else-Statement structure which can come after an if-statement. An else-statement will only execute it's code if the previous if-statement's condition was false.

If-Else-Statement Syntax

// Option 1 - Statement instead of Code Block
if (<expression>) <expression>;
else <expression>;
// Option 2 - Code Block
if (<expression>) 
{
    <statement>
    <statement>
    ...
}
else 
{
    <statement>
    <statement>
    ...
}

If-Else-Statement Example

int x = -20;
if (x > 0)
{
    println ("x is positive");
}
else
{
    println ("x is either zero or negative");   
}

Result:

x is either zero or negative

NOTE: You can only have an else-statement immediately after an if-statement. You will get an error if there is other code inbetween the if-statement and else-statement.

If-Elif-Statements (elif)

Sometimes it can be nice to make more than two choices. That's what Elif-Statements are for.

Like else-statements, elif-statements need to come immediately after an if-statement or another elif-statement. An elif-statement is mostly the same as an if-statement; It has a condition and a body of code that will execute if that condition is true. If an if-statement's condition is false, then the elif-statement's condition is checked.

Elif-statements can be chained together off of the same if-statement. An else-statement can also come after an elif-statement, however, an elif-statement cannot come after an else-statement. An else-statement can only be the last in a conditional chain.

If-Elif-Else-Statement Syntax

// Option 1 - Statement instead of Code Block
if (<expression>) <expression>;
elif (<expression>) <expression>;
elif (<expression>) <expression>;
//... there can be unlimited elif-statements
else <expression>;
// Option 2 - Code Block
if (<expression>) 
{
    <statement>
    <statement>
    ...
}
elif (<expression>)
{
    <statement>
    <statement>
    ...
}
elif (<expression>)
{
    <statement>
    <statement>
    ...
}
//... many more elif-statements
else 
{
    <statement>
    <statement>
    ...
}

If-Elif-Else-Statement Example 1

int x = -20;
if (x > 0) // false; -20 is less than 0
{
    println ("x is positive");
}
elif (x == 0) // false; -20 does not equal 0
{
    println ("x is zero");   
}
// this else runs if none of the above ifs/elifs run
else
{
    println ("x is negative");
}

Result:

x is negative

If-Elif-Else-Statement Example 2

int x = 0;
if (x > 0) // false; 0 is not greater than 0
{
    println ("x is positive");
}
elif (x == 0) // true; 0 == 0
{
    println ("x is zero");   
}
// this else does not run because an above elif-statement was true
else
{
    println ("x is negative");
}

Result:

x is zero

Nested If-Statements

Since if-statements can have statements in their body of code, that means you can have if-statements inside if-statements! This is called Nesting.

Nesting if-statements can allow for more complex choices to be made based on many conditions.

Nesting Example 1

int x = -20;
if (x > 0) // false; -20 is not greater than 0
{
    println ("x is positive");
}
// this else runs if x <= 0
else
{
    // nested if-else-statement
    if (x == 0)
        println ("x is zero");
    else
        println ("x is negative");
}

Result:

x is negative

If-statements can also be nested with For-Statements and While-Statements (See Loops).
