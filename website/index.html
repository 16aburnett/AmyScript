AmyScript - Programming Language

AmyScript is a middle-to-high-level programming language that is similar to C, C++, Java, and Javascript. This language and its compilers/interpreters are a part of a passion project and are not fully supported. Therefore, I do not recommend using this language commercially.

AmyScript is still currently in development. The expectation is that AmyScript is compiled to AmyAssembly which will then be interpreted. AmyAssembly is a lower-level assembly-like language that can be executed with the AmyAssembly Interpreter (Also in development).

Below is an example program written in AmyScript.

AmyScript

function int add (int a, int b)
{
    return a + b;
}

int c = add (7, 5);

print ("7 + 5 = ");
printInt (c);
println ("");

You can then run the code with the AmyAssembly Interpreter and you will get the output below.

Console Output

As you can see, it can be very tedious to write code in AmyAssembly, but the AmyScript code is significantly shorter.
