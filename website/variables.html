Variables are a crucial component for programming. They allow us to store values to use them later in arithmetic calculations, loops, printing, etc. They can also allow us to reference a value that might be inputted by a user.

This section goes over how to declare, assign, and access variables and their values.

Declaring Variables

Variables have 2 crucial components: The type and the value. AmyScript is statically typed which means when you declare variables, you have to specify the variable's type. See below:

Variable Declaration Syntax

<type-specifier> <variable-name>;

Variable Declaration Example

int x;

int denotes that the variable x stores an integer value

Assigning Variables to Values

Variables can store values through assignment statements. Assignment statements use assignment operators to store a value in a given variable.

Variable Assignment Syntax

<type-specifier> <variable-name> = <expression>;

Variable Assignment Example

int x = 7;

x is assigned the value of 7. When x is used in arithmetic expressions or other contexts, it will be treated as a 7 until it is reassigned.

Variables can be reassigned to new values later on.

Variable Reassignment Syntax

<variable-name> = <expression>;

The variable's type only needs to be specified the first time that the variable is referenced.

Variable Reassignment Example

int x = 7;
x = 16;

x is reassigned to the value of 16.

NOTE: Variables must be assigned to a value with a matching type. A variable's type cannot change.

Valid Variable Assignments

int a = 7;            // int var assigned to 7 (int)
float b = 3.14;       // float var assigned to 3.14 (float)
char[] str = "text";  // char[] var assigned to string literal
int[] l = new int[5]; // int[] var assigned to an int array allocator expression

Invalid Variable Assignments

int a = 6.24;       // Error: 6.24 is a float
float b = "string"; // Error: float cannot be assigned a char[]
int[] l = 10;       // Error: int[] cannot be assigned an int

Accessing Variable Values

Variables can be accessed through the variable's name.

Accessing Variable Example

int a = 7;
int b = 3;
int c = a * b;

Variables a and b are assigned to 7 and 3 respectively, and c is assigned to the result of the multiplication of a and b (which is 7 * 3 = 21).

Certain variables, like objects and arrays, need the use of other operators to access certain values in the variable.

Accessing from an Array

int[] nums = new int[4];
// use the subscript operator to assign values to specific indexes
nums[0] = 1;
nums[1] = 42;
nums[2] = 17;
// use the subscript operator to look up the value at a specific index
nums[3] = nums[0] + nums[1]; // 1 + 42 = 43

Accessing from an Object

class Person
{
    public field char[] name;
    private field int   age;
    constructor (char[] name, int age)
    {
        this.name = name;
        this.age = age;
    }
    public method int getAge ()
    {
        return this.age;
    }
}
Person person = new Person ("Amy", 22);
// use dot operator to access fields 
person.name;      // returns "Amy"
// use dot operator to access methods 
// methods are functions so you also need the parentheses and any arguments
person.getAge();  // returns 22 (int)
