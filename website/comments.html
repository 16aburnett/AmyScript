AmyScript - Comments

Comments are a very important component of programming languages. Comments allow you to annotate your code and provide explanations to what/how/why a piece of code works. Comments allow you to give information to other programmers that are working on the same program or can give information to yourself when you return to a part of the program.

Comments are skipped over by the compiler/interpreter and don't change anything about the code.

Currently, AmyScript only has Single-line Comments.

Single-Line Comments (//)

Single-line comments in AmyScript are similar to C++, Java, and Javascript. Single-line comments are denoted by // and span over the rest of that line, until a newline or end-of-file is reached. Everything from the // to the newline or end-of-file will be ignored by the computer, so you can write anything there to give information about the code around it.

Single-line Comment Syntax

//<text>

Single-line Comment Example

// This is a single-line comment
// The below function adds two given integers together and returns the result
function int add (int a, int b)
{
    // returning the result
    return a + b;
}
